2. Études de Cas Opérationnelles

Étude de Cas #1 : Anticipation (Objectif 1)

DURÉE : 1h30

Contexte : Intégration de 10 intérimaires en période de pic d'activité (Printemps).

Données Techniques :

  • Chariots : R489 Cat 1A, 3 et 5 en circulation simultanée.
  • Stockage : Masse et Racks (Hauteur 10m).
  • Risque majeur : Collision piéton/engin en zone de picking.

Travail à faire :

Établissez le Plan d'Accueil Sécurité spécifique à cette zone. Comment régulez-vous les flux pour isoler les piétons des engins sans stopper la production ?

Étude de Cas #2 : Crise & Management (Objectif 2)

DURÉE : 1h30

Scénario : Accident réel - Choc chariot élévateur contre préparateur (Quai n°4).

ALERTE !

Le blessé est au sol, conscient mais se plaint d'une vive douleur. L'équipe s'attroupe. Le planning de départ des camions est serré.

Livrables attendus :

  1. Action immédiate : Rédigez votre protocole PAS (Protéger, Alerter, Secourir).
  2. Communication Descendante : Rédigez le message de remobilisation pour l'équipe afin d'éviter le "climat d'insécurité".
  3. Communication Ascendante : Rédigez le SMS/Email flash pour votre Directeur de Site.

Étude de Cas #3 : Analyse & English (Objectif 3 et 4)

DURÉE : 2h00

Analyse technique (Arbre des causes) :

Identifiez les causes racines :

  • Matériel : Alarme de recul HS.
  • Humain : Pas de gilet haute visibilité porté.
  • Organisation : Retard de 20 min sur le chargement.

Reporting Anglais (ECF S3) :

Translate and summarize for HQ :

"Explain the accident, the root cause (equipment failure + PPE missing) and your immediate corrective action plan."
